## 開発環境
- ruby 2.5.3
- rails 6.0.2.1
- postgresql 11.6

## 仕様
https://chiharakenta.github.io/unipos-clone_api-document/

## セットアップ手順

```
$ git clone https://github.com/chiharakenta/unipos-clone-api.git
```

```
$ cd unipos-clone-api
```

```
$ bundle install --path vendor/bundle
```

```
$ rails s
```
